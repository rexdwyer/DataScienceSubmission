We can see there are almost 29,000 incidents (not all crimes.)

The more interesting features are perhaps: Category (e.g., "LARCENY"), DayOfWeek (e.g., "MONDAY"), Time (e.g., "14:28"), and PdDistrict (e.g., "TENDERLOIN").

Let's see if some days of the week have more incidents than others:

We can see that more incidents occur during the weekend than on workdays. A pie chart would not be appropriate for this data because angles are not perceived as easily as lengths; thus, a bar chart. We can easily see that there are roughly 10% more incidents on Fridays than on Tuesdays. (NB: It is possible that there are more Fridays than Tuesday in the dataset. I should have checked this.)

Now let's look by hour. The time is given in 24-hr format.

We can see a relatively smooth pattern, except that there is a large bump at the noon hour. It seems possible that 12:00 is inserted as a placeholder in some records, so let's investigate that further. This time we record incidents by minute in the day. There are 24 * 60 = 1440 minutes.

The white-collar crimes extortion, fraud, and embezzlement seem to be disproportionately reported at noon. It appears that loitering is reported disproportionately at noon; however, there are only three incidents of loitering in the data set. Likewise, there is only one incident of bribery. alldist = data["PdDistrict"]
distlist = sorted(alldist.value_counts().keys().tolist())
fraudcounts = {s:0 for s in distlist}
for d in fraud["PdDistrict"]: fraudcounts[d] += 1
fraudcounts = [fraudcounts[d] for d in distlist]
prostcounts = {s:0 for s in distlist}
for d in prost["PdDistrict"]: prostcounts[d] += 1
prostcounts = [prostcounts[d] for d in distlist]

N = len(distlist)

ind = np.arange(N)  # the x locations for the groups
width = 0.35       # the width of the bars

fig, ax = plt.subplots()
rects1 = ax.bar(ind, fraudcounts, width, color='r')

rects2 = ax.bar(ind + width, prostcounts, width, color='g')
# add some text for labels, title and axes ticks
ax.set_ylabel('Incidents')
ax.set_title('Fraud and Prostitution by District')
ax.set_xticks(ind + width)
ax.set_xticklabels(distlist,rotation=90)

ax.legend((rects1[0], rects2[0]), ('Fraud', 'Prostitution'))


def autolabel(rects):
    # attach some text labels
    for rect in rects:
        height = rect.get_height()
        ax.text(rect.get_x() + rect.get_width()/2., 1.05*height,
                '%d' % int(height),
                ha='center', va='bottom')

autolabel(rects1)
autolabel(rects2)

plt.show()

There is no obvious relationship between drug and prostitution incidents. The distinct with the most prostitution incidents is somewhere in the middle for drug incidents. Note a difficulty with this graph: The prostution scale runs only to 67, while the drugs scale approaches 300. A remedy could be to have two scales, one on each side of the chart, to make the heights of the drug bars similar to the prostitution bars.
